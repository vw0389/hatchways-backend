# Hatchways Backend

Simple json API that requests data from another api and serves it up


Written using:
- Nodejs
- Express
- axios
- apicache

Tested using:
- chai
- chaihttp
- mocha

## To Run
Go to root directory of hatchways-backend that directly contains the package.json file
```
npm i
```
### Starting the API
```
npm start
```
### Tests
```
npm test
```
